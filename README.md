# DMV Students Performance Analysis

## Overview
This project analyzes the Students Performance dataset using RStudio.  
The objective is to perform statistical analysis and data visualization using only R.

## Dataset
Source: Kaggle – Students Performance in Exams  
The dataset contains student scores in:
- Math
- Reading
- Writing  

It also includes:
- Gender
- Race/Ethnicity
- Parental level of education
- Lunch type
- Test preparation course

## Tools Used
- R
- RStudio
- tidyverse
- ggplot2

## Analysis Performed
- Data loading and structure inspection
- Missing value check
- Creation of total score variable
- Gender-wise performance analysis
- Test preparation impact analysis
- Parental education impact
- Correlation analysis between subjects
- Data visualization using histograms, boxplots, and scatter plots

## Key Insights
- Students who completed the test preparation course scored higher on average.
- Reading scores are generally higher than math scores.
- There is strong positive correlation between math, reading, and writing scores.
- Parental education level has a noticeable impact on total score.
